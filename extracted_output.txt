📋 Especificaciones Técnicas para Google Antigravity
1. El Objetivo del Proyecto
"Desarrollar una Web App de Inteligencia de Negocios para la evaluación de 60 campos petroleros en Venezuela. El propósito es permitir a inversionistas identificar objetivos de inversión mediante análisis comparativo de datos técnicos, financieros y de riesgo."
2. Stack Tecnológico Requerido
Lenguaje: Python 3.x.
Framework de UI: Streamlit.
Gestión de Datos: Pandas para procesamiento en memoria / SQLAlchemy para conexión a base de datos.
Seguridad: Implementar streamlit-authenticator (basado en YAML o DB) para control de acceso por roles (Admin/Inversionista).
Control de Versiones: Repositorio en GitHub.
3. Modelo de Datos (Esquema de Variables)
Indica a Antigravity que debe crear una estructura que soporte los siguientes grupos de datos:
Identificación: Campo, Bloque, Tipo de Desarrollo.
Reservas: Líquido (MMBN), Gas (MMMPC), Petróleo Equivalente (MMBN).
Producción: Prod. Acumulada, RGP, Producción Actual (BD/Mpcd).
Geología/Yacimiento: Gravedad API, Gravedad Esp. Gas, Permeabilidad (K), Profundidad de referencia.
Factores de Riesgo: Disponibilidad de Recurso Humano, Infraestructura, Inseguridad, Madurez Tecnológica.
4. Lógica de Visualización (Dashboards)
Solicita que el código generado incluya estas vistas específicas:
Vista de Portafolio: Un mapa interactivo (usando pydeck o folium) donde el tamaño de las burbujas represente las Reservas de Petróleo Equivalente.
Análisis de Sensibilidad: Un "Cross-plot" interactivo donde el eje X sea el % de Hidrocarburo Liviano-Mediano y el eje Y sea la Producción Actual.
Radar de Riesgo: Un gráfico de radar para cada campo que promedie: Infraestructura, Seguridad y Talento Humano.
5. Funcionalidades de Usuario
Módulo Admin: Formulario interactivo para añadir "Más campos" o "Más tipos de información" (columnas dinámicas).
Exportación: Botón para generar reportes en PDF/Excel de los campos seleccionados por el inversionista.
Filtros Globales: Filtros por Bloque, por rango de Gravedad API (Liviano/Mediano/Pesado) y por nivel de madurez tecnológica.
🛡️ Instrucciones de Seguridad para el Código
"Genera un módulo de gestión de secretos (st.secrets) para manejar las credenciales de la base de datos y las llaves de encriptación. El acceso debe estar restringido mediante un login inicial obligatorio antes de renderizar cualquier dato del sector petrolero." Esto lo podemos lograra como en el caso de otras aplicaciones que ya hemos desarrollado, apoyados en un documento de control en formato Google Sheet, y que guardaremos en Google Drive.
💡 Recomendación de Impacto
Para que el inversionista tome decisiones rápidas, podemos calcular automáticamente un "Score de Atractividad". Se Puede definir así:
Score = (Reservas * 0.4) + (Infraestructura * 0.3) - (Inseguridad * 0.3)
Este número simple permitirá ordenar los 60 campos de mayor a menor interés en un solo clic.
